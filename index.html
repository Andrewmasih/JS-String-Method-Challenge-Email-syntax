String Method Challenge

we currently have a list of users full-names and domain names and top level domains (eg. com, net, co.uk)

Challenge criteria is to use: string methods, string concatenation and a string interpolation. Challenge is to use the slice() and toLowercase methods along with string concatenation on the three defined varibles to define and assign a value to the 'emailAddress' varible, so that it meets the requirements. Requirement 1 the first and last name should be lowercase, and seperated by a dot. 2 the domain name should be seperated from the first and last name and the tld by an @ symbol. 3 the tld should be seprated from the domain name with a dot
